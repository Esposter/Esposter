# Infrastructure v9 Roadmap

Development stateful naming wave.

## Scope

Move development stateful services and externally referenced endpoints to the official naming convention after v8 succeeds. This wave should also migrate Pulumi resources to their final parent hierarchy wherever possible by creating target resources with the correct `parent` option from the start.

## Pulumi Notes

The v8 Logic Apps, action groups, event subscriptions, and Logic App role assignments were created without a Pulumi `parent` because their natural parent resources (`d-shp-rg-esposter-auea-001`, `d-shp-func-esposter-auea-001`, `d-shp-evgt-esposter-auea-001`) were still at legacy names at the time. Adding a legacy parent and then migrating it would require two state migrations instead of one.

Set `parent` on each v8 resource in the same wave that migrates its parent, using the create/cutover/delete pattern (`pulumi import` + `pulumi state delete`):

| Resource                                                   | `parent` to set          | When                       |
| ---------------------------------------------------------- | ------------------------ | -------------------------- |
| `dev-logic-esposter-auea-001..004`                         | `devRgEsposterAuea001`   | Legacy RG decommission     |
| `dev-ag-esposter-auea-001..003`                            | `devRgEsposterAuea001`   | Legacy RG decommission     |
| `dev-evgs-esposter-auea-001..002`                          | `devEvgtEsposterAuea001` | Event Grid topic migration |
| Website Contributor role assignments (logic-001, 002, 004) | `devFuncEsposterAuea001` | Function App migration     |
| EventGrid Contributor role assignments (logic-003, 004)    | `devEvgtEsposterAuea001` | Event Grid topic migration |

## Rollback

- Restore development app and function environment values to legacy endpoints and keys.
- Disable target Event Grid subscriptions if they are delivering duplicate events.
- Keep legacy storage, search, function, Web PubSub, Event Grid, and monitoring resources intact until the rollback window closes.
