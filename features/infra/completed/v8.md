## Pulumi Notes

- `parameterValueType: "Alternative"` is NOT in the Pulumi azure-native v3 `ApiConnectionDefinitionPropertiesArgs` schema — it is silently dropped.
- Must be set after each connection create/recreate via: `az rest --method PUT --uri "https://management.azure.com/subscriptions/{sub}/resourceGroups/{rg}/providers/Microsoft.Web/connections/{name}?api-version=2016-06-01" --body '{"location":"australiaeast","properties":{"api":{"id":"..."},"displayName":"...","parameterValueType":"Alternative"},"tags":{"Application":"Esposter"}}'`
- Pulumi's `PUT` on connection updates does NOT clear `parameterValueType` — Azure preserves it as a separate field.

After v8 completed, the `apicn` token was revised to `apic` (CAF-compatible derivation: API + C(onnection), matching the `apim` pattern). The `bdg` token was also revised to `bgt` (budget consonant extraction). Budget resources are still in legacy `dShpBdg...` format and will use `bgt` when eventually migrated to target naming.
