## Rollback

- Restore production app and function environment values to legacy endpoints and keys.
- Restore `packages/app/package.json` asset generation URL if the target storage account fails.
- Disable target Event Grid subscriptions if duplicate delivery appears.
- Keep legacy production storage, search, function, Web PubSub, Event Grid, and monitoring resources intact until the rollback window closes.
