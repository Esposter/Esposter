## Work Items

- [ ] Re-run live inventory and compare against Pulumi state.
- [ ] Confirm v8 and v9 are complete, documented, and preview-clean.
- [ ] Create target-named production action groups with identical receivers and `protect: true`.
- [ ] Update production Smart Detector and budget references to target action groups.
- [ ] Add target-named production API connections and Logic Apps.
- [ ] Prepare target production Event Grid subscriptions but keep duplicate delivery disabled until cutover.
- [ ] Run `pnpm infra:preview` and verify only v10 production low-risk resources are created or updated.
- [ ] Apply v10 in a low-traffic window.
- [ ] Verify action group receivers, Logic App runs, and Event Grid delivery behavior.
- [ ] Remove legacy v10 production resources in a separate cleanup apply after verification.

## Rollback

- Point Smart Detector rules and budgets back to legacy action groups.
- Disable target Event Grid subscriptions if duplicate delivery appears.
- Re-enable legacy Logic Apps if target workflows fail.
