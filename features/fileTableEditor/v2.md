# File Table Editor — Feature Roadmap v2

## UX / Workflow

- [ ] **Global search** ⚡ — a single search box filtering rows where any column matches; `v-data-table` has a `search` prop that may handle this with near-zero wiring
- [ ] **Column freeze/pin** ⚡ — toggle per column to mark it `fixed: true` so it stays visible while scrolling horizontally; Vuetify supports this natively, just needs a toggle in the column action row (view-only config, no command needed)

## Data Quality & Cleaning

- [ ] **Fill down** — fill null/empty cells in a selected column with the last non-null value above; same pattern as `NullStrategyCommand` (new command with full undo/redo support, targets a single column)
- [ ] **Regex find & replace** — extend the existing find & replace with a regex toggle; infrastructure already exists

## Column Enhancements

- [ ] **Computed/derived columns** — define a formula (e.g., `price * quantity`) that generates a read-only column; requires a formula parser

## Out of Scope

- ~~**Row grouping** — pushes toward Excel territory; out of scope for the platform's casual nature~~
- ~~**Conditional formatting** — highlight cells based on value conditions; out of scope for the platform's casual nature~~
- ~~**Cell validation rules** — per-column validation (e.g., "must be positive"); out of scope for the platform's casual nature~~
