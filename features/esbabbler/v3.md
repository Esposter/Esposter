## Moderation System

Spec: [`specs/moderation.md`](specs/moderation.md)

### Schema

- [x] `timeoutUntil` column on `usersToRooms` — nullable timestamp; migration
- [x] `bans` Drizzle schema (`roomId`, `userId`, `bannedBy`, `bannedAt`) + migration

### Server

- [x] `AdminActionType` enum — `shared/models/message/AdminActionType.ts`
- [x] `moderationRouter` — `server/trpc/routers/message/moderation.ts`
  - [x] `executeAdminAction({ roomId, targetUserId, type, durationMs? })` — permission gate per action type; check `isManageable`
  - [x] `onAdminAction({ roomId })` — subscription; targeted `userId` receives action
  - [x] `readModerationLog({ roomId, cursor })` — cursor-paginated; behind `ManageRoom`
- [ ] Timeout enforcement — reject in `createMessage` (and `forwardMessage`) if `timeoutUntil > now()`
- [x] Ban cascade — delete `usersToRooms` + `usersToRoomRoles`; insert into `bans`; block join/invite for banned users
- [x] `readBans` / `unbanUser` procedures — behind `BanMembers`
- [x] Moderation log write — append to Azure Table `ModerationLog` (`partitionKey: roomId`, `rowKey: reverseTickedTimestamp`, fields: `type`, `actorId`, `targetId`, `durationMs?`)

### Client

- [x] Force-mute / force-unmute handler — `useAdminActionMap`; sets `isForceMuted`, disables local toggle
- [x] Kick from voice handler — `useAdminActionMap`; calls `leaveVoice()` + snackbar
- [x] Kick from room handler — `useAdminActionMap`; calls `storeDeleteRoom()` + snackbar
- [x] "Audit Log" tab in room settings — `AuditLog/Index.vue`; behind `ManageRoom`

### Tests

- [x] `server/trpc/routers/message/moderation.test.ts` — all action types, permission gates, `isManageable` check, `onAdminAction` subscription, `readBans`, `deleteBan`, `readModerationLog`
