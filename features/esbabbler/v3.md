## Moderation System

Spec: [`specs/moderation.md`](specs/moderation.md)

### Schema

- [ ] `timeoutUntil` column on `usersToRooms` — nullable timestamp; migration
- [ ] `bans` Drizzle schema (`roomId`, `userId`, `bannedBy`, `bannedAt`) + migration

### Server

- [ ] `AdminActionType` enum — `shared/models/message/AdminActionType.ts`
- [ ] `moderationRouter` — `server/trpc/routers/moderation.ts`
  - [ ] `executeAdminAction({ roomId, targetUserId, type, durationMs? })` — permission gate per action type; check `isManageable`
  - [ ] `onAdminAction({ roomId })` — subscription; targeted `userId` receives action
  - [ ] `readModerationLog({ roomId, cursor })` — cursor-paginated; behind `ManageRoom`
- [ ] Timeout enforcement — reject in `createMessage` (and `forwardMessage`) if `timeoutUntil > now()`
- [ ] Ban cascade — delete `usersToRooms` + `usersToRoomRoles`; insert into `bans`; block join/invite for banned users
- [ ] `readBans` / `unbanUser` procedures — behind `BanMembers`
- [ ] Moderation log write — append to Azure Table `ModerationLog` (`partitionKey: roomId`, `rowKey: reverseTickedTimestamp`, fields: `type`, `actorId`, `targetId`, `durationMs?`)

### Client

- [ ] Force-mute / force-unmute handler — voice store; `isMuted = true`, disable local toggle until `ForceUnmute`
- [ ] Kick from voice handler — call `leaveVoice()`; show snackbar
- [ ] Kick from room handler — navigate away; server deletes `usersToRooms`
- [ ] "Audit Log" tab in room settings — cursor-paginated log; behind `ManageRoom`

### Tests

- [ ] `server/trpc/routers/moderation.test.ts` — all action types, permission gates, `isManageable` check, `onAdminAction` subscription, `readModerationLog`
