# Starred / Saved / Bookmarked Messages

Per-user bookmarks stored per `userId/roomId/rowKey`, with a dedicated "Saved" view in the sidebar.

## Why not

- Room-wide **pins** already cover the "save this message" use case.
- Per-user personal collections fit a Slack/email model, not a room-based chat platform.
- The practical need is met by shipped features: **Copy Message Link** + jump-to-message, and **`/remind`**.
