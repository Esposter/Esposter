# Esbabbler — Call Session Migration (Phase 1b → 1c)

Decoupled calls from room-scoped identity. Enables shareable meeting links (like Google Meet / Microsoft Teams).

---

## Current State

### `callSessionsInMessage` table

Persistent 1:1 anchor per room. Session survives server restarts; participants do not.

```ts
callSessionsInMessage = pgTable(
  "call_sessions",
  {
    id: text().primaryKey(), // 12-char alphanumeric; the id IS the shareable token
    roomId: uuid().references(() => roomsInMessage.id, { onDelete: "cascade" }),
  },
  { schema: messageSchema },
);
```

### `invitesInMessage.token`

`INVITE_TOKEN_LENGTH = 8`. Both invite and call session IDs generated by `createToken(length)`.

### `createToken(length)` — cryptographically secure

Uses `crypto.getRandomValues` with a 62-char alphanumeric set.

```ts
export const createToken = (length: number): string => {
  const bytes = crypto.getRandomValues(new Uint8Array(length));
  return Array.from(bytes, (byte) => TOKEN_CHARACTERS.charAt(byte % TOKEN_CHARACTERS.length)).join("");
};
```

### Procedure naming (`server/trpc/routers/room/call.ts`)

| Procedure           | Auth             | Input                        | Notes                                                                                 |
| ------------------- | ---------------- | ---------------------------- | ------------------------------------------------------------------------------------- |
| `readCallSessionId` | member           | `{ roomId }`                 | Reads session row; returns `id` string (`""` if none exists)                          |
| `joinCallByRoomId`  | member           | `{ roomId }`                 | Creates session if needed (3-retry inline); returns `{ callSessionId, participants }` |
| `joinCall`          | authed (no room) | `{ id }`                     | Joins by session id; no room membership needed                                        |
| `leaveCall`         | authed (no room) | `{ callSessionId }`          |                                                                                       |
| `setMute`           | authed (no room) | `{ callSessionId, isMuted }` |                                                                                       |
| `sendSignal`        | authed (no room) | `{ callSessionId, payload }` |                                                                                       |
| `onJoinCall` etc.   | authed (no room) | `callSessionId` (12-char)    | All subscriptions take callSessionId, not roomId                                      |

`requireCallSession(db, callSessionId)` — verifies session exists before allowing subscription.

### `readInviteToken` and `readCallSessionId` services

Both return `string` with `""` as the empty sentinel — never `null`.

### Client store (`app/store/message/room/call.ts`)

| Ref                          | Purpose                                                         |
| ---------------------------- | --------------------------------------------------------------- |
| `activeCallSessionId`        | Session the user is IN — drives `leaveCall`, `setMute`, signals |
| `currentRoomCallSessionId`   | Session for the VIEWED room — set by `useCallSubscribables`     |
| `callRoomId`                 | Room ID of active call — only for admin action room checks      |
| `callSessionParticipantsMap` | `Map<callSessionId, CallParticipant[]>`                         |

Store methods:

- `joinCallByRoomId()` — joins by current room (calls `$trpc.roomCall.joinCallByRoomId`)
- `joinCall(id)` — joins by session id (calls `$trpc.roomCall.joinCall`)

### `useCallSubscribables` flow

1. Room entry → `readCallSessionId({ roomId })` → returns `id` string
2. `setCurrentRoomCallSessionId(callSessionId)` — if `""`, return early (no session yet)
3. `readCallParticipants({ callSessionId })` → `setParticipants`
4. If already in call: re-join via `joinCallByRoomId()` (uses `currentRoomCallSessionId` already set)
5. Subscribe `onJoinCall / onLeaveCall / onSetMute` with `callSessionId`
6. Room exit → `leaveCall.mutate({ callSessionId })`, `setCurrentRoomCallSessionId("")`, `cleanupAll()`

### Shareable call link

`/call/[id]` page calls `joinCall({ id })`.
Any authenticated user with the id can join regardless of room membership.
